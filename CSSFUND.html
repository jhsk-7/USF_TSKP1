Conceptual Exercise: CSS Fundamentals

Q1: Explain how CSS differs from HTML in the context of web development.

HTML sets the contents of the page. CSS styles the contents of the page.

Q2: List the methods of including CSS in an HTML document and briefly describe each.

  1. Can create a style sheet with extension ".css" and link it to an html document by using a link element in the header of your html page.
  2. Can create styles directly in the head of your html document.
  3. Can create styles directly in a line of your html document
  4. Can createa link to a non operating system style by calling the href and then calling the style in your style sheet.

Q3: Describe the anatomy of a CSS rule.

  • Reference element or elements separated by comma
  • Reference an ID using "#" symbol followed by the id name. Example #username
  • Reference a class using "." symbol followed by the class name. Example .text-feedback
  • Then create curly brackets with different properties for the css style.

Q4: When might you want to use RGBA instead of RGB?

RGBA will allow you to set the transparency.

Q5: How do Hex color values differ from RGB?

Hex uses numbers and letters to ID a color. RGB uses three values for the amount of "red", "green" and "blue" needed to create a specific color.

Q6: When might a developer prefer to use HSL over other color formats?

When they want to control the saturation and lightness.

Q7: What are the primary text properties used in CSS to modify the appearance and layout of text?

font-size, font-weight, font-family, line-height

Q8: In what scenarios might it be beneficial to use vh or vw as a unit for font size?

Q9: Explain the difference between em and rem units.

em is font size relative to it's parent. rem is font size relative to the root of the page.

Q10: If multiple font families are listed in the font-family property, how does the browser decide which one to display?

In the order that the font was listed.
